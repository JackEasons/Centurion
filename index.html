Centurion

Built using SASS and CSS3 media queries, Centurion is a responsive web framework that scales with your device. No longer do you need to worry about the screen size of an iPhone or an Android tablet since Centurion does the work for you.

Want to test Centurion out simply fork or download the repository on Github to get started and see the power a responsive framework can have on your next web project.

CSS3/HTML 5

Built-on CSS3 and HTML5 semantics to allow you to start captializing on a next generation standard.

Responsive

No matter the device: Mac, PC, iPhone, iPad, iPad mini, or Android tablet, Centurion does the work of making your web project responsive and viewable on multiple devices

Prototype Quick

Need to built a web wireframe quick, look no further than Centurion. Scalable and easy to learn syntax makes building a prototype simple.
