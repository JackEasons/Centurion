Need a reason to use Centurion?

Latest Code

Built on CSS3 and HTML5 semantics to allow you to start capitalizing on a next generation standard. If you're worried about browser support we recommend the html5shiv to add the basic support for semantic coding.

It's Responsive

No matter the device: Mac, PC, iPhone, iPad, iPad mini, or Android tablet, Centurion does the work of making your web project responsive and viewable on multiple devices. Using some basic magic it even works in IE7.

Quickly Prototype

Need to built a web wireframe quick? Look no further than Centurion with it's scalable and easy syntax to get a new prototype up fast. Download the code or fork the repo on Github to get started.

What can Centurion do?

Responsive

Take a seat, build your desktop or mobile version, and watch your web site scale up or down instantly. A responsive grid built with the same heart as the 960 grid, but conceived using percentages rather than strict columns.

Semantic or Unsemantic

Since Centurion is built using Sass we've allowed developers to use the baked in grid system or the ability to create their own semantic grid. Whatever your code style Centurion is built to accommodate whatever is thrown at it.

Integration

Most of frameworks only work with one Sass mixin library. However, Centurion has been redesigned to alongside Compass, Bourbon, and many different libraries. If you have an issue with your mixin library, then submit an issue for us.

Easy to Learn

Finding it hard to learn a new framework? Don't worry anymore about a bloated code base with markup you have to override with important tags. Centurion features easy to read markup, classes and ID's that are semantic or unsemantic for your personal web tastes, and provides some basic documentation to get you up to speed.

Optimized

We've spent hours tweaking and fine-tuning the code in Sass to deliver a minimal footprint for web development. No more bloated code that is hard to understand and slows your browser down. If you use a Sass precompiler we've made it easy to remove code that you don't need.

Customizable

Comes out of the box with custom jQuery plugins for navigation (desktop and mobile), tables, tab boxes, alerts, and tooltips. Since Centurion was built using Sass you can even customize the core files to fit your needs and make your own custom version.

Still need convincing?

If you are unfamiliar with the gambit of options, check out our documentation on Centurion for syntax and class structures.

We're always interested to see how the community will use Centurion. Drop us a line and let us know how and where you used Centurion so that we can showcase the many ways Centurion is making the web better.
