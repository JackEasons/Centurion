The documentation for Centurion is currently being written using Github. Please refer to that documentation for any information that you might need in utlizing Centurion on your next web project.

A Small List of What It Can Do

  • Custom CSS buttons, no images required.
  • Grid dimensions from 960 px down to 320 px wide.
  • Image scaling.
  • Heading scales based on size of browser. (Use em, instead of px)
  • Navigation for Desktop and Mobile
  • HTML 5 framework with backwards compatibility to IE6
